# U-create Database Services ( UDS )

UDS is a platform to manage **Postgresql** services e.g ( manage users, database).

## About Project

- Language: PHP ( >= 7.1.3)
- Framework: Laravel ( >= 5.7 )
- Database: PostgreSql
